# 1. 说明


实现
* 从文件读入生成图
* 添加边 ([非]连通)
* 打印 顶点 - 相连接的边
* 计算v的度数
* 计算所有顶点的最大度数
* 计算所有顶点的平均度数
* 计算自环的个数
* DepthFirstSearch 单点连通性 (递归)
* DepthFirstPaths 深度优先搜索路径(递归)
* BreadthFirstPaths 广度优先搜索路径(队列)
* CC 深度优先搜索检测连通性
* Cycle 检测给定的图是否是无环图 (不存在自环或平行边)
